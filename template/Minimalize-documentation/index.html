Folder Structure

  • style.css - main style.css
  • The css folder - contains the CSS for all major browsers, prettyphoto, responsive to site makes responsive for all devices and the font awesome (Please ensure that your new styles are applied. Also make sure that they carry enough "weight" and there isn't a style lower in the CSS file that is being applied after yours.)
  • The fonts folder - contains the font awesome font files.
  • The images folder - contains all the images that are being used in the stylesheets for the layout purpose and it has a subfolder like blog-images, portfolio-images, which contains images used in posts, pages and other areas
  • The video folder - contains the Video and Audio files used in the template.
  • The content folder - contains blog and portfolio items, which comes while clicking the Load more option in Portfolio and while Scrolling in Blog Detail Page.
  • The js folder - contains javascript files and plugins as follows,
    1. Modernizr (modernizr-2.6.2.min.js)
    2. Masonary Portfolio (masonry.pkgd.min.js.js)
    3. Validation plugin for form validation (jquery.validate.min.js)
    4. ToolTip (jquery.tipTip.minified.js)
    5. Tabs (jquery.tabs.min.js)
    6. Sticky Navigation (jquery.sticky.js)
    7. MeanMenu plugin for responsive menu (jquery.meanmenu.min.js)
    8. Isotope plugin (jquery.isotope.min.js)
    9. Google Map plugin (jquery.gmap.min.js)
    10. CarouFredSel plugin (jquery.carouFredSel-6.2.0-packed.js)
    11. BX Slider (jquery.bxslider.min)
    12. Pushy Menu (pushy.min.js)
    13. Vertical Nav icon (jquery.visualNav.min.js)
    14. For Slidebars (slidebars.js)
    15. For Scroll Plugins
      1. greensock.js
      2. ScrollToPlugin.min.js
    16. For Revolution Slider Plugins
      1. jquery.themepunch.revolution.min.js
      2. jquery.themepunch.tools.min.js
    17. Combined js in (plugins.min.js)
    18. And custom js (custom.js)

We have used Google Fonts (Source Sans Pro, Lato , Pacifico ) in our template. You can use any Google font simply as following,

1. Please replace the following lines in all layout files.

            
				@import url(http://fonts.googleapis.com/css?family=Roboto:400,100,300,500,700);

2. Find the following lines in style.css and replace with your new font family.

 
				body { font-weight:normal; font-size:13px; background:#fff; color:#353535; font-weight:300;  font-family: 'Roboto', sans-serif;}
